If you want to generate a new image, place the content of your Runtime_ESBSE folder here. 
This folder is obtained when downloading Talend along with the Open Studio. 
Example location: /home/user/TOS_ESB-20211109_1610-V8.0.1/Runtime_ESBSE

Its typical structure looks like this:

activemq  bin  docs  lib  LICENSE.txt  zookeeper
add-ons   container  examples  licenses  NOTICE
